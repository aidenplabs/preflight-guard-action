# Security Policy

This repository is the GitHub Action distribution surface for Preflight Guard.

## Reporting

- Do not post secrets, tokens, private keys, database URLs, or production incident details in public issues
- If you believe there is a security-sensitive problem in this repository, report it privately to the maintainers instead of opening a public issue

If the concern is about the underlying scanner logic rather than the action wrapper, include that context in the private report so it can be routed to the main project as needed.
